# BFVReadPlayerMessage

## 原作者的软件描述

A software to read BFV server player send's message which can find long message's sender id.  
version v1.

在我们玩战地5的时候经常看到有人刷屏并且没有id，打开游戏后运行**BFVReadPlayerMessage.exe**，就可以获取ID
